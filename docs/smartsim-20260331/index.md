# SmartSim, Utrecht, March 31, 2026

**[Registration is via the LUMI web site.](https://lumi-supercomputer.eu/events/lumi-smartsim-workshop-march2026/)**


-   Location: The Netherlands, Utrecht, 
    [Office Park Hoog Overborch (level Hoog Catharijne), Moreelsepark 48, room 2.03](https://maps.app.goo.gl/e8udMkyVfZbhGTJLA).
